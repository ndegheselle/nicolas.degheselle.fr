Open-source framework built on top of Vue.js, designed for creating server-side rendered (SSR) and statically generated web applications.

## Feedback

- SSR is forced—unnecessary for simple SPAs.
- SSR adds complexity in managing state between server and client (check authentification).
- Auto-imports are useful but enforce rigid folder structures (no nesting).
