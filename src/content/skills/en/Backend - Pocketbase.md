---
title: "Pocketbase"
group: "Web backend"
level: 1
---

Open-source backend that provides a built-in database, authentication, and real-time APIs in a single lightweight executable.

- Very flexible: Can be used as a framework to add functionalities (new routes, handle events, etc.).
- Uses a SQLite database in the background: It would be great if different databases could be supported.

Overall very nice solution, easy to setup, extend and use.
